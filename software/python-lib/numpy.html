numpy¶

numpy 是一个用c实现的python库，用来进行大规模数组、矩阵计算, 众所周知， python的循环迭代非常慢， 所以在进行大量计算时，可以使用numpy库来优化

相关地址: http://wiki.scipy.org/Tentative_NumPy_Tutorial
