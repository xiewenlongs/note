Ivy is not avaliable

Ivy是ant的一个插件， 报这个错说明Ivy插件没有安装，或者没有配置好.

第一步，从官网下载Ivy tgz包并下载编译成jar包， 然后把Ivy.jar 拷贝到/usr/share/ant/lib/ 下

第二步, 配置环境变量 ANT_HOME, export ANT_HOME=/usr/share/ant/
