**Ivy is not avaliable**

Ivy是ant的一个插件， 报这个错说明Ivy插件没有安装，或者没有配置好. 解决方法:

    1. 从官网下载Ivy tgz包并下载编译成jar包， 然后把Ivy.jar 拷贝到/usr/share/ant/lib/ 下
    2. 配置环境变量 **export ANT_HOME=/usr/share/ant/**
